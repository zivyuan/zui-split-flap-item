## 🍐 快速上手：

```html
<zui-split-flap-item :char="A" />
```



## 🍊 参数

####  🍒 char: `string | number`

单个字符

####  🍒 size: `number`

组件尺寸

####  🍒 color: `string`

文字颜色。暂未实现


####  🍒 font: `string`

字符字体。

默认："Robot", "monospace", "DIN Condensed"

####  🍒 delay: `number`

翻牌动画延迟

单位：毫秒

默认：0
